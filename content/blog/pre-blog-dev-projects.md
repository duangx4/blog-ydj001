## 07.14 — 全线开工

### 微信聊天记录解密

34 万条聊天记录，分散在 19 个 SQLite 数据库中。

先尝试 PyWxDump，但微信 4.0 改了进程名（`Weixin.exe`），不兼容。换用 `WeChatDataAnalysis`，内存扫描模式提取密钥，19 个库全部解密成功，343MB 数据零错误。

> 工具选型决定了效率——第一个方案不行不要死磕，换工具比修兼容快得多。

### 个人画像 + 证书归档

从聊天记录提取了完整个人履历：学业、竞赛、证书、社交特征。顺手配了 Tesseract OCR，截图识别证书：蓝桥杯省二、科普科幻国三、模拟飞行优胜奖等，全部归档。

同时建立了**三线职业战略**：

| 优先级 | 方向 | 目标 |
|--------|------|------|
| 🥇 一线 | 电网无人机巡检 | 电气 + CAAC + 电工证 → 县局差异化选手 |
| 🥈 二线 | PLC + 工业自动化 | 珠三角设备厂/系统集成商 |
| 🛡️ 保底 | CAAC 无人机飞手 | 持证随时上岗 |

### IMA 知识库 — 5 门课程灌入

把大一课程框架写进 IMA 知识库，方便随时检索：

- 电路分析基础（8 章 + 核心公式）
- 高等数学 Ⅰ/Ⅱ（12 章框架）
- 大学物理 Ⅰ（力学/电磁/热学）
- 线性代数
- C 语言程序设计（语法 + 嵌入式关联）

### Agent 体系搭建

创建了主 Agent **实力人老大**和子 Agent **电气实力人**（电气 RAG 助理），配置了多 Agent 调度权限和人设规范。这是整个工作流的大脑。

## 07.22 — 基础设施扩军

### QQ 机器人三件套部署

架构：`EasyBot ↔ LLBot(QQ协议端) ↔ EasyBot-MCDR(MC服插件)`

部署方式很折腾：Windows 拉 Docker 镜像 → `docker save` 导出 tar → SCP 传到云服务器 → `docker load` → `docker compose` 启动。镜像一共 1.2GB，传了挺久。

端口映射：5000(WebUI) / 3081(管理面板) / 26990(WebSocket)

### 服务器运维基线

0 号机（山东枣庄，NAT 内网）承担了全部业务：

| 服务 | 说明 |
|------|------|
| OpenClaw | Agent 运行环境 |
| MCSManager | MC 服务器面板 + 守护进程 |
| EasyBot + LLBot | QQ 机器人双容器 |
| MCDR | MC 服务端插件系统 |

同时配置了本地 ↔ 云端的同步脚本，人设文件双向同步。

## 07.23 — 博客 + 双机联动

### 新增 1 号机 — 香港 VPS

2 核 2G / 30G SSD / 20Mbps 无限流量 / 独立 IPv4。买来就是为了免备案 + 博客 + FRP 隧道。

### FRP 隧道 — 打通内网管理

1 号机有公网 IP，0 号机没有。在 1 号机装了 frps，0 号机装 frpc，建立了 4 条隧道：

```
bot.ydj001.xyz      → LLBot 管理面板
easybot.ydj001.xyz  → EasyBot WebUI
mcsm.ydj001.xyz     → MCSManager
frp.ydj001.xyz      → FRP 管理面板
```

所有后台统一走子域名，不用记端口号，4 条隧道加起来不到 3MB 内存。

### Nginx + SSL + DNS

Nginx 1.24，Certbot 自动 SSL，Cloudflare DNS 托管 + 代理。防火墙配置时差点把自己锁在门外——UFW 忘了先开 22 端口，血的教训。

### Hugo 博客上线

建站本身反而是最轻松的。Hugo + PaperMod，一个二进制构建 82ms。配合之前搭的整套基础设施，发布流程压缩到一行命令。
